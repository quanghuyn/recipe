# Recipe

Get recipe for your meal

## Live Demo

Website: [https://recipe-88552.netlify.app/]



## Tech stack

- HTML/SASS/JavaScript


## Feature

- App use model MVC
- Find a recipe dependent on your keyword 
- Wire your recipe
- Save recipe into bookmarks 
- Custom servings in recipe
